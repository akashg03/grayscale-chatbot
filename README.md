# Grayscale Chatbot

A minimal, accessible chatbot-style UI that maps the last two digits of a number to a grayscale background color for the bot’s reply.

**Architecture:** Next.js App Router in `app/`, UI in `components/`, color logic in `lib/`, types in `types/`, unit tests in `tests/`, and documentation in `docs/`.

## Quick start

```bash
npm install
npm run dev
```

## Deploy (Vercel)

```bash
npm run build
vercel --prod
```

## Docs

See the [docs/](docs/) folder for architecture, color logic, accessibility, testing, and future considerations.

## Tools

Next.js, TailwindCSS, TypeScript, Vitest, Vercel, AI coding assistant.
